== Description ==

The coming soon with Darth Vader you were only dreaming of. Simply activate the plugin and anyone that is not currently logged into  your site will see it! 

It hides your site when you are building it, as long as you are logged in as admin, you would be the only one able to see the site.

Come to the dark side! We have cookies!

###  Features:

* Works with any WordPress Theme
* Free Coming soon page template
* Darth Vader
* Easy to activate

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3.All done!
